# heart-disease-project
This project focuses on analyzing and predicting heart disease mortality using 
machine learning techniques. The dataset was collected from Data.gov and includes 
records from the years 2015, 2016, and 2017. The data was first combined and then 
cleaned to remove missing and unnecessary values. 
Exploratory Data Analysis was performed using different graphs to understand the 
distribution and patterns in the data. After preprocessing, both regression and 
classification models were applied to the dataset. Several models such as Linear 
Regression, Random Forest, XGBoost, Logistic Regression, and KNN were used. 
The performance of these models was evaluated using different metrics like R² score, 
accuracy, precision, recall, and F1-score. Among all the models, Random Forest 
showed the best performance in both regression and classification tasks. 
The main aim of this project is to understand how machine learning can be used to 
analyze healthcare data and make better predictions. The results show that these 
techniques can be useful in identifying patterns and supporting decision-making in 
the healthcare sector.
